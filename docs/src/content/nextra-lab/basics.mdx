## Steps

<Steps>
### Install docs dependencies

Install `nextra` and `nextra-theme-docs` in `docs/package.json`.

### Configure app router integration

Use `docs/src/app/[[...mdxPath]]/page.tsx` with `importPage`.

### Add content and metadata

Build section indexes with `_meta.ts` + `index.mdx`.
</Steps>
